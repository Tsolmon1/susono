Ажлуулах:
================================================================================
1. NodeJS суулгана.
2. Сервер дээр хурдан хуулахын тулд инсталл хийгээгүй байгаа. 
	Тийм учраас прожектийн фолдер дотроос cmd-гээс "npm install --production" гэж суулгаж ажлуулна шүү.

Ямар ч хамаагүй энгийн HTTP сервер дээр тавьж ажлуулна.
АНХААРУУЛГА: Шууд HTML файлийг веб хөтөч дээр нээж ажлуулвал асуудал алдаанууд гарах магдлалтай.


НЭМЭЛТ: 
Хэрвээ NodeJS суулгасан бол "npm install -g http-server" гэсэн коммандыг ажлуулж сервер суулгаж болно.
Суулгасаны дараа прожектийн фолдер дотроос "http-server -a localhost -p 8000" гэсэн коммандыг ажлуулж серверээ асааж болно.

Хөгжүүлэлт:
================================================================================
1. NodeJS суулгана.
2. Суусан бол прожектийн фолдер дотор cmd-гээс "npm install --only=dev" гэдэг комманд ажлуулж хөгжүүлэлтийн орчинг суулгана.
3. Дараа нь прожектийн хэрэглэгдэж буй library-нуудыг "npm install" гэж суулгана.(Hard-луу)
	Хэрвээ node_modules фолдер хоосон болсон нөхцөлд л уншуулна уу.
4. Одоо хөгжүүлэлт хийж болно. Хийгдсэн IDE: Visual Studio Code.
5. Ажлуулахад прожектийн фолдер дотороос "npm start" коммандыг ажлуулна.
